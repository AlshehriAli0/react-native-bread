# Example App

A demo app showcasing all features of `react-native-bread`.

## Prerequisites

- Bun
- Expo CLI
- Xcode (for iOS) or Android Studio (for Android)

## Getting Started

1. `cd ..`
2. `bun install` (from monorepo root)
3. `cd example`
4. `npx expo prebuild`
5. `bun run ios` or `bun run android`

## Available Scripts

- `bun start`: Start the Expo dev server
- `bun run ios`: Build and run on iOS simulator
- `bun run android`: Build and run on Android emulator
- `bun run web`: Start the web version
